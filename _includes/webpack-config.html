Webpack conf plugin

webpack-config-plugin is a webpack configuration plugin that helps you merge one or more configuration files together with environment variables.

I took a few hours of my leisure time to build a simple branding and website.

THE PROBLEM

A simple "how it works" was needed, and decided to design a fast new website. Serving the purpose to help developers to understand and trust the plugin even more.

As designers, we should be pushing more open design processes. I firmly believe in open source software, where is way more needed now than ever.

THE BRANDING

A logo that would meant a plugin and using similary with the webpack actual branding.

THE PROCESS

A few screens that will be animated in order to simply explain this plugin.
